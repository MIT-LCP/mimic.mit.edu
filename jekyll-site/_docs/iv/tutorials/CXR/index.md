# Chest X-Ray (CXR) Tutorials

Tutorials for working with the MIMIC-CXR module, which contains chest X-ray images and radiology reports.

## What You'll Learn

These tutorials cover:

- Accessing and loading chest X-ray images
- Working with radiology reports
- Image preprocessing and normalization
- Combining imaging data with clinical information
- Deep learning approaches for medical imaging
- Quality assessment and filtering

## Prerequisites

Before starting these tutorials:

1. Obtain access to MIMIC-CXR dataset
2. Familiarize yourself with the [CXR Module documentation]({{ "/docs/iv/modules/cxr/" | relative_url }})
3. Have Python environment set up with imaging libraries (PIL, OpenCV, etc.)

## Tutorial Structure

Each tutorial includes:
- Learning objectives
- Code examples
- Sample outputs
- Exercises for practice
- Links to relevant research papers

The tutorials progress from basic image loading to advanced analysis techniques suitable for research publications.
