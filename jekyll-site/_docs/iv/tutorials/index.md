# MIMIC-IV Tutorials

Step-by-step tutorials for working with MIMIC-IV data across different modules and use cases.

## Available Tutorials

### Chest X-Ray (CXR) Tutorials
Learn how to work with medical imaging data in the MIMIC-CXR module.

### Waveform Tutorials  
Tutorials for processing and analyzing physiological waveform data including ECG signals.

## Getting Started

These tutorials provide practical examples of:

- Data extraction and preprocessing
- Common analysis patterns
- Visualization techniques
- Research methodologies
- Best practices for working with clinical data

Each tutorial includes code examples, sample queries, and explanations of the underlying clinical concepts.

For beginners, we recommend starting with the [Getting Started]({{ "/docs/gettingstarted/" | relative_url }}) section to understand how to access the data, then exploring the tutorials relevant to your research interests.
