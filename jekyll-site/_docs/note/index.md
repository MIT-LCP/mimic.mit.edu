# Note Module

The Note module contains deidentified free-text clinical notes for hospitalized patients. This includes:

- Progress notes
- Discharge summaries  
- Nursing notes
- Physician notes
- Radiology reports
- Other clinical documentation

## Tables in this Module

The Note module contains:

- **discharge** - Discharge summaries
- **radiology** - Radiology reports
- **physician** - Physician notes
- **nursing** - Nursing documentation
- **general** - General clinical notes
- **social_work** - Social work notes
- **case_management** - Case management notes
- **pharmacy** - Pharmacy notes
- **nutrition** - Nutrition notes
- **respiratory** - Respiratory therapy notes

All notes have been deidentified to protect patient privacy while preserving clinical content for research.
