# MIMIC Databases

The MIMIC database family includes several datasets, each designed for different research purposes and containing different types of clinical data.

## Database Versions

### 🏥 MIMIC-IV
The latest and most comprehensive version of MIMIC, containing data from 2008-2019.

[Explore MIMIC-IV →](/docs/iv/)

### 🏥 MIMIC-III  
The previous version of MIMIC, containing data from 2001-2012.

[Explore MIMIC-III →](/docs/iii/)

### 🏥 MIMIC-II
The original MIMIC database, containing data from 2001-2008.

[Explore MIMIC-II →](/docs/ii/)

## Specialized Modules

### 🫁 Chest X-Ray (CXR)
Chest radiographs and associated radiology reports.

[CXR Module →](/docs/cxr/)

### 🚑 Emergency Department (ED)
Emergency department stays and associated clinical data.

[ED Module →](/docs/ed/)

### 📝 Clinical Notes
Free-text clinical notes from various hospital departments.

[Notes Module →](/docs/note/)

## Key Features

- **De-identified**: All data has been de-identified to protect patient privacy
- **Comprehensive**: Includes demographics, vital signs, laboratory results, medications, and more
- **Longitudinal**: Follows patients across multiple hospital stays
- **Validated**: Data has been extensively validated and cleaned
- **Open Access**: Freely available for research after credentialing

---

{: .highlight }
> **Choosing a Database**: Most new research should use MIMIC-IV as it contains the most recent and comprehensive data. MIMIC-III remains available for reproducibility and comparison studies.
