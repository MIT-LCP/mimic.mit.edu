## About MIMIC

MIMIC is a large, publicly available critical care database containing de-identified health data associated with patients admitted to critical care units at a large tertiary care hospital. The database includes information such as demographics, vital sign measurements, laboratory tests, medications, and more.

## MIMIC Modules

MIMIC-IV is separated into "modules" to reflect the provenance of the data:

### MIMIC-IV-Core
{: .d-inline-block }

New
{: .label .label-green }

Patient demographics, admission tracking, and stay information is available in MIMIC-Core.

### MIMIC-IV-Hosp
{: .d-inline-block }

Hospital
{: .label .label-blue }

The Hosp module provides all data acquired from the hospital wide electronic health record. This includes laboratory measurements, microbiology cultures, medication information, services provided, billed diagnoses and procedures, and so on.

### MIMIC-IV-ICU
{: .d-inline-block }

ICU
{: .label .label-purple }

The ICU module contains information collected from the clinical information system used within the ICU. This includes highly granular information such as hour-to-hour vital signs, information about fluid management, and other charted observations.

### MIMIC-IV-ED
{: .d-inline-block }

Emergency
{: .label .label-red }

The ED module contains data for emergency department patients including a triage assessment, nurse-validated vital signs, medicine reconciliation, and treatment information.

### MIMIC-IV-CXR
{: .d-inline-block }

Imaging
{: .label .label-yellow }

MIMIC-cxr provides chest x-ray images and radiology reports for a subset of patients admitted to the emergency department.

### MIMIC-IV-Note
{: .d-inline-block }

Clinical Notes
{: .label .label-grey }

The Note module contains deidentified free-text clinical notes for hospitalized patients.

### MIMIC-III
{: .d-inline-block }

Legacy
{: .label .label-yellow }

MIMIC-III is an older version of MIMIC. It contains an older group of patients (ending in 2012), and a subset of the ICU and hospital information available in MIMIC-IV. We highly recommend researchers starting new studies to use the above modules in MIMIC-IV.

## Getting Started

1. **Become a credentialed user** on PhysioNet by completing training in human subjects research
2. **Sign the data use agreement** for the datasets you need
3. **Access the data** through cloud services or local download
